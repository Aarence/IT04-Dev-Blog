Week 2 Dev Blog

HTML Semantics

  1. Header:

    2. Typically used for navigation or introductory content, the <header> element can be placed at the top of a page to encapsulate navigation menus and logos.

  2. Article:

    3. The <article> tag is often used for content like news articles, blog posts, or independent pieces of content. It contains one or more heading tags (<h1> to <h;6>).

  3. Nav:

    4. The <nav> element is used for navigation menus and is specifically designed for key navigation elements within the site, not for every link on the page.

  4. Aside:

    5. Reserved for content that is indirectly related to the main content, the <aside> tag is commonly found in sidebars or call-out boxes.

  5. Section:

    6. The <section> element is used to group related content together. It is particularly useful for breaking content into distinct sections within an article tag.

  6. Footer:

    7. Just as the <header> encapsulates introductory content, the <footer> element is used for the footer section of a page, which often contains copyright information and additional navigation links.

CSS Basics

  1. Color:

    2. You can set the color of text, backgrounds, and borders using CSS. For example, color: green; will change the text color to green.

  2. Size:

    3. CSS allows you to control the size of elements, such as font size, width, and height, to create a visually appealing layout.

  3. Position:

    4. You can use CSS to position elements on your page, determining where they appear in relation to other elements.


Ways to Includde Styles in HTML

There are different methods to apply CSS styles to your HTML content:

  1. Inline Style:

    2. You can apply styles directly to individual HTML elements using the style attribute. For example:
    p style="color: green;".

  2. Internal Stylesheet:

    3. You can define your styles within the style element tag in the HTML file, affecting multiple elements within the document. For instance:
    <style>
    * {
    color: green;
    }
    </style>

  3. External Stylesheet:

    4. To maintain a clear separation between HTML and CSS, you can create an external CSS file (e.g., basic.css) and link it to your HTML document using the <link> element. This is a more organized approach, especially for larger projects
    <link rel="stylesheet" href="basic.css" type="text/css" />

CSS Selectors

  1. Attribute Selectors:

    2. Select elements based on their attributes, e.g., input[type="text"].

  2. Class and ID Selectors:

    3. Apply styles to elements with specific classes or IDs, like .my-class and #my-id..

  3. Pseudo-Classes:

    4. Style anchor elements depending on their state, like a:link, a:visited, a:hover, a:active, and a:focus.
